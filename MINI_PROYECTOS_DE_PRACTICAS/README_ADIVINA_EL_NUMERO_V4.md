# ADIVINA EL NÚMERO V4

Mini proyecto desarrollado en Python como práctica de:

- Variables
- Condicionales
- Bucles while
- Diccionarios
- Contadores
- Lógica de programación
- Gestión de estados del juego

## Características

- Sistema de 3 niveles
- Números aleatorios
- Vidas limitadas por nivel
- Registro de partidas ganadas
- Registro de partidas perdidas
- Registro del nivel máximo alcanzado
- Menú interactivo
- Estadísticas del jugador

## Lo que aprendí

Durante esta versión practiqué:

- Corrección de bugs
- Manejo de contadores
- Uso de variables booleanas
- Uso de diccionarios
- Depuración paso a paso
- Diseño de lógica para videojuegos simples
